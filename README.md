# Employee Management Dashboard

A modern, responsive Employee Management Dashboard built with React.js and Bootstrap.

## Features

- **Authentication**: Secure login/logout functionality
- **Dashboard**: Summary statistics (Total, Active, Inactive employees)
- **Employee Management**:
  - Add new employees with form validation
  - Edit existing employee details
  - Delete employees (with confirmation)
  - Toggle employee active/inactive status
- **Image Upload**: Profile image upload with preview
- **Search & Filter**:
  - Search by employee name
  - Filter by gender
  - Filter by active/inactive status
  - Combined filtering capabilities
- **Print Functionality**: Print employee lists
- **Responsive Design**: Works on desktop, tablet, and mobile

## Tech Stack

- **Frontend**: React.js (JavaScript)
- **UI Framework**: Bootstrap 5 + React Bootstrap
- **Icons**: Font Awesome
- **Routing**: React Router DOM
- **Printing**: React-to-print
- **State Management**: React Context API
